# bluray-burner-scripts

"WORK IN PROGRESS" - "PROOF OF CONCEPT".

These scripts don't work out-of-the-box. They require editing to make them work on your system. Copy them to /usr/local/bin or anywhere else in your path, then run blud and insert a bluray disc. Wait till the gnome-terminal pops up. OR without blud, just run blustart after inserting a disc and waiting a moment.

If you get an input/output error when copying discs during the EXTRACTION of the disc then don't burn it, it will probably break your disc!

My Bluray Hardware is an "External USB TechPulse120 Bluray Burner".

Enjoy!
