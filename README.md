# bluray-burner-scripts

These scripts don't work out-of-the-box. They require editing to make them work on your system. Copy them to /usr/local/bin or anywhere else in your path, then run blud and insert a bluray disc. Wait till the gnome-terminal pops up.

My Bluray Hardware is an "External USB TechPulse120 Bluray Burner".

![Screenshot](https://github.com/gtoyman/bluray-burner-scripts/blob/main/TechPulse120.jpg)

Enjoy!
